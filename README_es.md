# Protocolos para Mixomicetos

🌐 [English](README.md)

Protocolos para el cultivo, identificación y caracterización del microbioma de plasmodios de mixomicetos, desarrollados en el **Laboratorio de Interacciones Bióticas, Centro de Investigación en Ciencias Biológicas, Universidad Autónoma de Tlaxcala, México**.

Estos protocolos fueron desarrollados y validados como parte del proyecto *Relación Amoebozoa–bacterias mediante análisis metagenómico del microbioma del plasmodio de mixomicetos* (SECIHTI CBF2023-2024-1480).

---

## Protocolos

| Protocolo | Descripción |
|-----------|-------------|
| [Cultivo en Cámara Húmeda](moist-chamber-culture.md) | Cultivo no estéril a partir de sustrato de campo; preserva la microbiota nativa (eubiosis) |
| [Cultivo en Agar](agar-culture.md) | Cultivo semi-axénico a partir de esporoforos; reduce la carga bacteriana (axenización / disbiosis) |
